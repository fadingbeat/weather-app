# WeatherApi — .NET 8 Web API

## Folder structure

WeatherApi/
├── Controllers/
│ ├── AuthController.cs # POST /api/auth/register, /api/auth/login
│ ├── WeatherController.cs # GET /api/weather/current, /api/weather/forecast
│ ├── SearchesController.cs # GET /api/searches
│ └── StatsController.cs # GET /api/stats
│
├── Services/
│ ├── IAuthService.cs + AuthService.cs
│ ├── IWeatherService.cs + WeatherService.cs # HttpClient proxy to OpenWeather
│ ├── ISearchService.cs + SearchService.cs # Save + retrieve searches
│ └── IStatsService.cs + StatsService.cs # Stats queries
│
├── Models/ # EF Core entities (DB tables)
│ ├── User.cs
│ └── SearchHistory.cs
│
├── DTOs/ # Request/response shapes (never expose EF models directly)
│ ├── Auth/
│ │ ├── RegisterRequest.cs
│ │ ├── LoginRequest.cs
│ │ └── AuthResponse.cs
│ ├── Weather/
│ │ ├── CurrentWeatherDto.cs
│ │ └── ForecastDto.cs
│ ├── Search/
│ │ └── SearchHistoryDto.cs
│ └── Stats/
│ └── StatsDto.cs
│
├── Data/
│ ├── AppDbContext.cs # EF Core DbContext
│ └── Migrations/ # Auto-generated by dotnet ef
│
├── Middleware/
│ └── ExceptionHandlerMiddleware.cs # Global error → consistent JSON response
│
├── appsettings.json # Non-secret config (committed)
├── appsettings.Development.json # Secrets — in .gitignore, never committed
└── Program.cs # Service registration + middleware pipeline
